# 풍력 AI 이중 제어 시스템 블럭도
```mermaid
flowchart TD
    A[과거 풍력발전기<br/>SCADA 데이터] --> C
    F[실측 풍속 센서] -->|실측 오차 보정| C
    F -->|실시간 오차 보완|E
    C[풍속 예측 AI] --> D[예측 풍속값]
    D --> E[최적 피치각 제어 AI]
    E --> G[최적 피치각]
    G --> I[피치 제어기]
```
